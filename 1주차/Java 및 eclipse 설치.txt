1. java 1.8 open_jdk 다운로드
2. c:드라이브의 Programfile폴더에 Java라는 이름의 폴더 생성과 그 안에 java 1.8 open_jdk폴더를 넣는다.
3.윈도우+break키 or 제어판-시스템의 고급시스템 설정 ->[고급]탭의 환경변수 설정 -> os(밑의 선택창)의 Path편집 -> java 1.8 open_jdk폴더의 bin의 경로를 삽입, 추가한다
(단 윈도우 7일경우 전체내용을 지우지말고 맨 뒤에 ;(세미콜론)을 붙인뒤 전체 경로 덧붙임)
4.이클립스 사이트로 간 뒤 download가 아닌 그 밑의 Package 클릭 -> 맞는 버전 다운로드
5.윈도우+r키 or 명령프롬프트 관리자 권한 실행 -> java -version과 javac -version을 입력시 에러가 나지않는다면 java환경변수 설정완료
6.이클립스 실행시 에러가 나지않는다면 완료
